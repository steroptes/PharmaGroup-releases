## 📋 Guide d'installation

### Étape 1 — Télécharger le fichier APK
Cliquez sur le lien de téléchargement ci-dessus ou rendez-vous sur la page [Releases](https://github.com/steroptes/PharmaGroup-releases/releases).

### Étape 2 — Autoriser l'installation
Votre téléphone vous demandera d'autoriser l'installation depuis des sources inconnues :
1. Allez dans **Paramètres** > **Sécurité** (ou **Applications**)
2. Activez **Sources inconnues** (ou **Autoriser l'installation d'applications inconnues** pour votre navigateur)
3. Sur Android 8+, vous pouvez autoriser uniquement votre navigateur (Chrome, Firefox, etc.)

### Étape 3 — Installer l'application
1. Ouvrez le fichier `PharmaGroup.apk` depuis vos **Téléchargements**
2. Appuyez sur **Installer**
3. Une fois l'installation terminée, appuyez sur **Ouvrir**

### Étape 4 — Se connecter
Utilisez vos identifiants PharmaGroup (email + mot de passe) pour vous connecter.
